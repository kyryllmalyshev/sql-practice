# SQL Practice

SQL queries written during structured IT Support preparation.
Covers real-world scenarios relevant to Application Support L2 roles.

## Topics Covered

- SELECT, WHERE, ORDER BY — filtering and sorting
- INNER JOIN, LEFT JOIN - single and multi-table queries
- Triple JOIN — connecting 3+ tables
- GROUP BY, HAVING — aggregations and filtering groups
- Subqueries — scalar, IN/NOT IN, nested
- CASE WHEN — conditional aggregations
- Complex queries — combining all of the above

## Tables Used

**users** — id, name, email  
**orders** — id, user_id, product_id, amount, status, created_at  
**products** — id, name, category, price  
**employees** — id, name, department, salary, hire_date  
**projects** — id, name, department, budget  

## Skills Demonstrated

- Troubleshooting data issues using SQL queries
- Analysing user behaviour and transaction patterns
- Writing efficient queries for support investigations
